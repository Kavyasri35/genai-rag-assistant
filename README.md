# GenAI RAG Assistant

This project is a **GenAI-powered Chat Assistant** built using **Retrieval-Augmented Generation (RAG)**.  
The assistant retrieves relevant information from documents and generates accurate responses.

## Features

- Document based question answering
- Retrieval-Augmented Generation (RAG)
- Embedding based similarity search
- Simple web chat interface
- Context based response generation

## Tech Stack

- Python
- Flask
- HTML
- Embeddings
- Vector similarity search
